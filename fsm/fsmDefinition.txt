    in1

    out1
Variables:
    int timeout = 5000
States:
state IDLE [ print(out1) ]
state ACTIVE [ print(out1) ]
state TIMING [ print('timing') ]

    IDLE --> ACTIVE: in1 [ tonumber(in1) == 1 ]
    ACTIVE --> TIMING: in1 [ tonumber(in1) == 0 ]

    TIMING --> IDLE : @ timeout
