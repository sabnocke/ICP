name WashingMachine
comment: Simulace pracího cyklu pračky: plnění, praní, máchání, odstřeďování, hotovo.
Input:
Output: status

Variables:
    int tIdle   = 1000
    int tFill   = 3000
    int tWash   = 5000
    int tRinse  = 4000
    int tSpin   = 2000
States:
    state IDLE    [ return status ]
    state FILL    [ return status ]
    state WASH    [ return status ]
    state RINSE   [ return status ]
    state SPIN    [ return status ]
    state DONE    [ return status ]
Transitions:
    IDLE    --> FILL  : @ tIdle
    FILL    --> WASH  : @ tFill
    WASH    --> RINSE : @ tWash
    RINSE   --> SPIN  : @ tRinse
    SPIN    --> DONE  : @ tSpin
    DONE    --> IDLE  : @ 2000
